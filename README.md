# Smart Email Reply Assistant

A professional Streamlit web app that classifies email intent and generates intelligent, context-aware responses using AI rules and NLP.

## Features
- **Intent Classification** — Auto-detects if an email is a Query, Complaint, Request, Scheduling, Follow-up, etc.
- **Smart Tone Selection** — Chooses Formal, Friendly, or Assertive tones depending on the sender context and sentiment
- **Professional Reponses** — Auto-generates structured, polite responses tailored to that intent
- **Response History** — Saves processing history and results for easy review

## Run Locally
```bash
pip install -r requirements.txt
streamlit run email_assistant_ui_pro.py
```

## Deployment
This app is configured to be deployed on **Streamlit Community Cloud**. Select `email_assistant_ui_pro.py` as the main file path during deployment.
